![](https://img.166.net/reunionpub/1_kol_20260113_d81dbe19ddb69deffcbe4082a4159e81)

**任务一：在禁阁第二层冥想 0/1**

完成方法：

点击坐下冥想并回答问题即可完成

 **位置：禁阁二层**

![](https://img.166.net/reunionpub/1_kol_20251229_cc0d1f8782e11a646a73ca76232425e6)

 **任务二：在圆梦村使用钓鱼竿钓一些好东西  0/1**

完成方法：

1.通过 遇境/云巢 的活动向导传送至宴会节活动地点

2.在冰面上使用钓竿钓到水母即可完成

![](https://img.166.net/reunionpub/1_kol_20251227_419577cf545a62c93b75b93f54a1e2aa)

![](https://img.166.net/reunionpub/1_kol_20251228_f1afb36fcf4cd7e1bfac1795d325685f)

 **任务三：点燃20根蜡烛 0/20**

完成方法：

1.点燃一堆大蜡烛

2.或点燃20根蜡烛即可完成

 **推荐地图：云野 霞谷 禁阁 遇境 云巢**

![](https://img.166.net/reunionpub/1_kol_20251112_44a17c81e9e66ee32253dc18232de5c2)
