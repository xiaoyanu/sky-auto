**任务一：向一位玩家鞠躬 0/1**

完成方法：

点击屏幕上方中间动作栏 面对玩家 点击并完成鞠躬动作即可

 **注：1～4级鞠躬动作均可完成；玩家包括陌生人与好友**

![](https://img.166.net/reunionpub/1_kol_20250707_802971fa279123e3ec30496503fe32f3.png)

 **先祖：礼貌学者 [站姿]**
