**任务一：向一位玩家鞠躬 0/1**

完成方法：

点击屏幕上方中间动作栏 面对玩家 点击并完成鞠躬动作即可

 **注：1～4级鞠躬动作均可完成；玩家包括陌生人与好友**

![](https://img.166.net/reunionpub/1_kol_20250612_16c409c30c3fc9dd6ef017ba19830f42.jpeg)

 **先祖：礼貌学士 [站姿]**
