**任务一：拯救一只被黑暗藤蔓困住的遥鲲 0/1**

完成方法1：

拿出心火蜡烛燃烧黑暗藤蔓拯救遥鲲即可完成

 **位置：**

 **雨林 密林遗迹**

 **1.密林遗迹中间亭子下方**

 **2.密林遗迹右侧矿洞前**

 **注：共有两处随机拯救一只即可完成**
