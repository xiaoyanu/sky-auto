# 光遇每日任务标题
![每日任务.001](https://img.166.net/reunionpub/kol_20260531_ba752bef32a12d3adaaea7999e2eee93)

**任务一：在圆梦村的温泉休息一下 0/1**

完成方法：

在圆梦村的温泉范围内等待60秒即可完成

 **位置：霞谷 圆梦村**

![每日任务.004](https://img.166.net/reunionpub/1_kol_20260531_c3f7c96ea80bc4f8ba01220c37e283ee)  
 **任务二：向一位朋友做个动作 0/1**

完成方法：

点击屏幕上方中间的表情菜单 面对好友 随机点击任意动作即可完成

**![](https://img.166.net/reunionpub/1_kol_20260317_363faabb5db8dd97edb1e382f5490936)**  
 **任务三：在荧光小菇旁恢复能量 0/1**

完成方法：

1.将斗篷能量全部消耗

2.靠近荧光小菇恢复能量即可完成

 **位置：雨林 荧光森林 密林遗迹 大树屋等**

![](https://img.166.net/reunionpub/1_kol_20260131_3b6ef780bd8a8479a93cfaa89b0a0751)

位置：霞谷 霞光城
