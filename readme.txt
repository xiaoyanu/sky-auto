# 光遇每日任务标题
![](https://img.166.net/reunionpub/1_kol_20250706_6fd7871527fe47b101bd6cff62ae8ae7.png)

**任务一：向一位朋友招手 0/1**

完成方法：

点击屏幕上方中间动作栏 找到招手动作 面对好友 点击并完成招手即可

 **注：1～6级招手动作均可完成**

![](https://img.166.net/reunionpub/1_kol_20250706_3f428516a51e7a91898ce91482206f99.png)

 **任务二：威严舰长在藏宝岛礁需要帮助 0/1**

完成方法：

1.前往藏宝岛礁找到 威严舰长 领取任务 [靠近岛屿，拍张照片]

2.根据指引来到岛屿旁使用留影相机拍照即可完成

 **位置：暮土 藏宝岛礁**

![](https://img.166.net/reunionpub/1_kol_20250706_eaecd45525a6e7c94aa42626226dc4fe.jpeg)

![](https://img.166.net/reunionpub/1_kol_20250706_0524526be651bb737dfd2858050ad22f.jpeg)

 **任务三：点燃20根蜡烛 0/20**

完成方法：

1.点燃一堆大蜡烛

2.或点燃20根蜡烛即可完成

 **推荐地图：云野 霞谷 禁阁 遇境 云巢**

![](https://img.166.net/reunionpub/1_kol_20250706_38869eb0c74ae0091b6c8dfd42ab21d5.jpeg)

 **任务四：在远古战场中央冥想 0/1**

点击坐下冥想并回答问题即可完成

 **位置：暮土 远古战场**
