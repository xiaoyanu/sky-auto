完成方法：

点击坐下冥想并回答问题即可完成

 **位置：禁阁二层**

![](https://img.166.net/reunionpub/1_kol_20260222_629bb0cf5bdb6a90e6c1d2c64ba7066c)

 **任务二：收集30点烛火 0/30**

完成方法：

1.领取任务

2.靠近地图当中 点燃的蜡烛上方的烛光/净化黑暗植物后的烛光 收集30滴即可

**![](https://img.166.net/reunionpub/1_kol_20251006_904146ea633d7507084b73346fe1b462.png)**
