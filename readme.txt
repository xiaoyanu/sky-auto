 **任务二：向一位朋友招手 0/1**

完成方法：

点击屏幕上方中间动作栏 找到招手动作 面对好友 点击并完成招手即可

 **注：1～6级招手动作均可完成**

![](https://img.166.net/reunionpub/1_kol_20250730_8f466e6da72fb5c62a83ef1c5f2e2418.png)
