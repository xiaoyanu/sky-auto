完成方法：

点击坐下冥想并回答问题即可完成

 **位置：霞谷 霞光城**

 **任务二：更换您的裤子 0/1**

完成方法：

1\. 前往 服饰神坛 裤子更换处

2\. 任意选择裤子 点击更换即可完成

 **任务三：向一位朋友做个动作 0/1**

完成方法：

点击屏幕上方中间的表情菜单 面对好友 随机点击任意动作即可完成
