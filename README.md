# 光遇每日任务标题
![每日任务.001](https://img.166.net/reunionpub/1_kol_20260529_d06e863e28cadf5c7089a528434edf34)

**任务一：帮助 嬉笑炮手或 犯困船匠在仙乡寻找宝藏 0/1  
** 完成方法：  
由寻宝节向导传送至活动地点，再由活动先祖传送至当日寻宝地点  
任选一位先祖完成寻宝即可

![每日任务.004](https://img.166.net/reunionpub/1_kol_20260516_4d1ce44ab38ddf601b89703118ea2237)

 **任务二：点亮一位玩家 0/1**

完成方法：

在天空王国中靠近任意其他玩家举起心火蜡烛等待对方点亮即可完成

![](https://img.166.net/reunionpub/1_kol_20260325_e71a9b6c837ab82c8945fb3274684bdc)

 **任务三：收集30点烛火 0/30**

完成方法：

1.领取任务

2.靠近地图当中 点燃的蜡烛上方的烛光/净化黑暗植物后的烛光 收集30滴即可

**![](https://img.166.net/reunionpub/1_kol_20251006_904146ea633d7507084b73346fe1b462.png)**
