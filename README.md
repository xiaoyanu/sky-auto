**任务一：向一位朋友招手 0/1**

完成方法：

点击屏幕上方中间动作栏 找到招手动作 面对好友 点击并完成招手即可

 **注：1～6级招手动作均可完成**

![](https://img.166.net/reunionpub/1_kol_20250703_272e811313748de19f5b68a6ef3a8ff9.png)

完成方法：

 **位置：云野 圣岛**

 **先祖：热血运动员 [动作]**

![](https://img.166.net/reunionpub/1_kol_20250703_bf293f2df4bf13b6b60140424f24f51b.png)
