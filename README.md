**任务一：点亮一位玩家 0/1**

完成方法：

在天空王国中靠近任意其他玩家举起心火蜡烛等待对方点亮即可完成

完成方法：

1.点燃一堆大蜡烛

2.或点燃20根蜡烛即可完成

 **推荐地图：云野 霞谷 禁阁 遇境 云巢**
