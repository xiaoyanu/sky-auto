完成方法：

1.将斗篷能量全部消耗

2.靠近水母恢复能量即可完成

 **云野 圣岛**

完成方法：

点击人物头部长按大叫即可掀翻螃蟹

 **推荐位置：**

 **云野 圣岛**

 **暮土 边陲荒漠/巨兽荒原**
